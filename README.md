# EconoLoop

A school-based community app where students can exchange unused or pre-loved items using a karma point system. Users can post items (e.g., flash drives, chargers) and assign a karma value, while others can request or redeem them if they have enough points. Karma is earned by participating in community events, encouraging engagement and sustainability. The platform promotes resource sharing, reduces waste, and strengthens student collaboration within the campus.
